### 🚀 Getting Started

#### 1. Clone the repository

```bash
git clone https://github.com/Rohan-Shridhar/page-construction.git
cd page-construction
````
#### 2. Install dependencies
```bash
npm install
````
#### 3. Run the project
```bash
npm run dev
````
