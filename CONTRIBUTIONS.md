<h1 align="center">🛠️ How To Begin Contributing</h1>

For anyone new to contributing, here is a quick guide:
1. Fork this repository
    - You can fork the repository by clicking "Fork" on the [main page](https://github.com/Rohan-Shridhar/page-construction) of the repository
2. Clone your fork
    - Use the command `git clone https://github.com/YOUR-USERNAME/page-construction`
3. Create a new branch
    - Use the command `git checkout -b YOUR-BRANCH-NAME`
4. Make your changes to the project
5. Stage, commit and push
    - Use the command `git add file-name` to stage files you have changed
    - Use the command `git commit -m "Insert a meaningful comment about your commit"` to commit
    - Use the command `git push origin YOUR-BRANCH-NAME` to push
6. Open a Pull Request to send in your contributions
    - Head to your fork on GitHub and open the Pull Requests tab to make a PR
    - Make sure to follow the PR template!

<h1 align="center">📋 PR Template</h1>
When submitting a PR, please include the following:

- Reference the issue number you have worked on
- Describe what changes you have made
- Explain clearly the purpose of these changes
- If applicable, describe how you tested your changes to ensure intended functionality
- If applicable, include screenshots for any UI changes

<h1 align="center">⚠️ Reporting an Issue</h1>

If you happen to find any unreported issues:
1. Open a [new Issue](https://github.com/Rohan-Shridhar/page-construction/issues/new)
2. Describe the issue clearly
    - What it is, how to reproduce or where the issue can be found
3. If applicable, add screenshots displaying the issue

<h1 align="center">🔄 Suggesting an Enhancement</h1>

Have any ideas for improvements?:
1. Open a [new Issue](https://github.com/Rohan-Shridhar/page-construction/issues/new)
2. Ensure it is titled as an Enhancement
3. Explain your idea and its purpose
